# Data-S-ralama
Bu proje, banka sıralama mantığını temel alan işlem yönetim modelini ele almaktadır. Veri yapısı olarak yığın (stack) kullanılarak infix ifadelerin postfix ve prefix biçimlerine dönüştürülmesi, işlem önceliği ve sıralama kurallarının uygulanması amaçlanmıştır. Çalışma, temel ifade çözümleme algoritmalarını göstermektedir.
